# DevOps App

## Description : 
Mini application Node.js réalisée dans le cadre d’un exercice DevOps.  
L’objectif est de mettre en place une pipeline CI/CD automatisée avec tests, build et analyse de sécurité.

---

## Fonctionnalités : 
- Serveur Node.js simple
- Tests unitaires
- Conteneurisation avec Docker
- Pipeline CI/CD avec GitHub Actions

---

## Lancer l'application en local : 
```bash
cd app
npm install
npm test
node src/index.js
